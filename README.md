Tool of OSINT and Dork hacking that work with Google and Duckduckgo


## Install

```bash
git clone https://github.com/Vault-Cyber-Security/getrails.git
cd getrails
pip3 install -r requirements.txt
python3 search.py 'term'
```
